# Quincalla

Quincalla is a minimal e-commerce site build with Laravel 5.1

## Installation

- Clone repository
- Run composer
```
composer install
```
- Migrations
```
php artisan migrate
```
- Seeds
```
php artisan db:seed
```

## Testing

- Unit
```
vendor/bin/phpspec run
```
- Functional
```
vendor/bin/phpunit
```
- Acceptance
```
vendor/bin/behat
```
